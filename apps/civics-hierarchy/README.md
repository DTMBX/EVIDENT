# Civics Hierarchy

Constitutional law reference platform mapping the hierarchy of legal authority from Natural Law through federal, state, and local jurisdictions.

Part of the [Evident Technologies](https://evident.icu) ecosystem.

## Tech Stack

- TypeScript, React 18, Vite
- Tailwind CSS, Radix UI, Lucide Icons
- localStorage-based KV persistence
- Hash-based client-side routing

## Development

```bash
npm install
npm run dev
```
