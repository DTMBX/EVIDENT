## Current Issues

1. **Mixed root level files** - 40+ files at project root
2. **Poor separation of concerns** - documentation scattered
3. **Test placement unclear** - tests in multiple locations
4. **Config files not organized** - 10+ dotfiles without grouping
5. **Deployment scripts at root** - Should be in scripts/infrastructure
6. **HTML files at root** - Should be in src/static or docs
7. **No clear frontend/backend separation** - All mixed together
8. **Unclear static assets location** - spread across multiple dirs

## Reorganization Steps

1. Create directory structure
2. Move files to appropriate locations
3. Update import paths if necessary
4. Update configuration references
5. Update CI/CD workflows
6. Verify all functionality
7. Commit with clear message
